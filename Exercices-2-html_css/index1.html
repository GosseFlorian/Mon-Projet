La vie d'ADA

Ada Lovelace, mathématicienne britannique du XIXe siècle, est considérée comme la première programmeuse de l'histoire pour avoir écrit le premier algorithme destiné à être executé par une machine, la machine analytique de Charles Babbage, ce qui en fait une pionnière visionnaire de l'informatique moderne.

Le nom de l'école Ada Tech School s'inspire de cette mathématicienne.

En quelques points :

  • Pionière de l'informatique : créa le premier algorithme pour ordinateur
  • mathématicienne visionnaire du 19e siècle
  • Collaboratrice de Babbage sur la machine analytique
  • Anticipait le potentiels des ordinateurs au-delà des simples calculs
